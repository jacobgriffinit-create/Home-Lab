# Home Lab

A personal home lab environment built to develop hands-on skills in network security, infrastructure, and cloud technologies.

## Projects

### Network-Wide DNS Filtering & VPN Exit Node

Deployed a network-wide DNS filtering solution using Pi-hole in a Docker container to block ads and malicious domains at the network level. Installed Tailscale directly on the Raspberry Pi configured as a VPN exit node enabling secure remote access to the home network from any location.

**Technologies used:** Raspberry Pi, Docker, Pi-hole, Tailscale, Linux

**Skills demonstrated:** Container deployment, DNS management, network security, VPN configuration, Linux administration
